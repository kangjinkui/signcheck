현재 이 세션에서 사용할 수 있는 스킬은 4개입니다.

  - init-docker-dangerous: Docker 컨테이너에서 Claude Code --dangerously-skip-permissions 모드 환경 초기화, 기능 구
    현, 결과를 MD로 저장
  - init-speckit: 새 프로젝트를 spec-kit 기반으로 초기화
  - skill-creator: 새 스킬 생성이나 기존 스킬 개선 가이드
  - skill-installer: 설치 가능한 스킬 조회, curated 스킬 설치, GitHub 저장소에서 스킬 설치

  원하면 다음도 바로 할 수 있습니다.

  - 특정 스킬의 SKILL.md 열어서 요약
  - 어떤 요청에 어떤 스킬을 써야 할지 추천
  - skill-installer로 추가 스킬 설치 진행
